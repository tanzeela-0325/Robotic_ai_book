# AI-Native Physical Humanoid Robotics Book

This repository contains the documentation and specifications for the AI-Native Physical Humanoid Robotics book project. The book covers the complete development of AI-native humanoid robots integrating ROS 2, simulation environments, AI cognition, and natural interaction systems.

## Project Structure

The documentation is organized into several key areas:

- **specs/**: Core specifications and implementation plans
- **docs/**: Docusaurus documentation files with frontmatter
- **src/**: Source code for the book's examples and implementations

## Documentation Structure

The documentation is structured around the following core modules:

1. **Feature Specification** - Complete requirements and user stories
2. **Quickstart Guide** - Getting started with the development environment
3. **Data Model** - Core entity definitions and relationships
4. **Research** - Foundational research and technical specifications
5. **Implementation Plan** - Detailed development roadmap
6. **Task Tracking** - Complete implementation tasks and progress

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run start
   ```
